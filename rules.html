Server Rules

Keep the server fair & friendly. Breaking rules can lead to mutes/bans.

  1. No cheating, hacked clients, or exploiting bugs.
  2. No harassment, hate speech, doxxing, or threats.
  3. Respect staff decisions. Use the proper channels for appeals.
  4. No advertising other servers without permission.
  5. Use common sense; if it ruins the fun for others, don’t do it.

Edit rules.html to change these.
